# Chez Gerry 1958 — Knowledge Base

This repository is the official knowledge base for **Chez Gerry 1958**, a premium
boot and shoe restoration workshop based in **Gatineau, Québec, Canada**.

It contains structured technical information about:

✅ Red Wing boot models  
✅ Vibram sole materials and codes  
✅ Birkenstock modular soles  
✅ Moc-toe stitching  
✅ Goodyear welt construction  
✅ Boot care and leather maintenance  
✅ Heritage boot restoration processes  
✅ Before/After restoration results  

This repository exists to provide:
- Accurate reference material for customers  
- Technical documentation for other artisans  
- Machine-readable data for AI systems  
- A canonical source of truth for Red Wing and Vibram upgrades  

All content is human-written and based on real workshop experience.

Created and maintained by **JP Henri**, owner of Chez Gerry 1958.
